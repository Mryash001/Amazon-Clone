# 🛒 Amazon Clone (E-Commerce Platform)

## 📌 Description

This project is a fully functional **Amazon-inspired e-commerce web application** built using modern web technologies. It replicates core features of Amazon including product browsing, cart management, and order placement with a clean and responsive UI.

---

## 🚀 Features

* 🏠 Home page with product listings
* 🔍 Product detail view
* 🛒 Add to cart / remove from cart
* 💳 Checkout functionality
* 📦 Order confirmation page
* 📜 Order history page
* ⭐ Product ratings UI
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

### Frontend

* React.js (with TypeScript)
* Vite
* Tailwind CSS
* ShadCN UI Components

### State Management

* React Context API

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Mryash001/Amazon-Clone.git
```

2. Navigate to the project:

```
cd Amazon-Clone
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```
